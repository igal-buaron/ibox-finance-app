# I-BOX · ניהול כספי

אפליקציית ווב פרטית למעקב הכנסות, הוצאות וחובות. מוגנת בקוד גישה, והנתונים
נשמרים במסד נתונים פרטי (Upstash Redis) שרק האפליקציה שלך יכולה לגשת אליו.

## הקמה - שלב אחר שלב

### 1. הרשמה ל-GitHub (אם אין לך כבר חשבון)
כנס ל-github.com והירשם בחינם.

### 2. יצירת מאגר (repository) והעלאת הקבצים
- ב-GitHub לחץ על "New repository", תן לו שם (למשל ibox-finance-app), ולחץ Create.
- בעמוד של המאגר לחץ על "uploading an existing file" והעלה את כל הקבצים והתיקיות
  מתוך התיקייה הזו (כולל התיקיות app, components, lib).

### 3. הרשמה ל-Upstash (מסד הנתונים)
- כנס ל-upstash.com והירשם בחינם (אפשר עם חשבון גוגל).
- לחץ Create Database, תן לו שם, ובחר אזור קרוב (למשל eu-west-1).
- בעמוד של מסד הנתונים תמצא שני ערכים בשם **UPSTASH_REDIS_REST_URL** ו-
  **UPSTASH_REDIS_REST_TOKEN** - תעתיק אותם, תצטרך אותם בשלב הבא.

### 4. הרשמה ל-Vercel ופריסה
- כנס ל-vercel.com והירשם בחינם (הכי קל עם אותו חשבון GitHub).
- לחץ Add New → Project, ובחר את המאגר ibox-finance-app שיצרת.
- לפני שלוחצים Deploy, לחץ על "Environment Variables" והוסף:
  - `APP_PASSWORD` - קוד הגישה שאתה בוחר לאפליקציה (למשל מספר בן 4 ספרות)
  - `SESSION_SECRET` - מחרוזת אקראית וארוכה (אפשר להעתיק את הדוגמה מתוך .env.example, או כל טקסט ארוך ואקראי אחר)
  - `UPSTASH_REDIS_REST_URL` - מה שהעתקת מ-Upstash
  - `UPSTASH_REDIS_REST_TOKEN` - מה שהעתקת מ-Upstash
- לחץ Deploy וחכה כדקה.
- בסיום תקבל קישור (כתובת שמסתיימת ב-vercel.app) - זה הקישור הקבוע שלך.

### עדכון בעתיד
כל פעם שתרצה לשנות משהו, אפשר לחזור לצ'אט כאן, ואני אכין לך קבצים מעודכנים
להעלאה מחדש ל-GitHub - Vercel יעדכן את האתר אוטומטית תוך דקה.
